Features

  • Query files in plain SPARQL 1.1, via the SERVICE <x-sparql-anything:> (see options)
  • Build knowledge graphs with CONSTRUCT queries, in plain SPARQL 1.1
  • Supported formats: XML, JSON, CSV, HTML, Excel, Text, Binary, EXIF, File System, Zip/Tar, Markdown, YAML, Bibtex, DOCx (see format-specific options)
  • Transforms files, inline content, or the output of an external command
  • Full fledged HTTP client to query Web APIs (headers, authentication, all methods supported)
  • Functions library for RDF sequences, strings, hashes, easy entity building, ...
  • Combine multiple SERVICE clauses into complex data integration queries (thanks to SPARQL)
  • Query templates (using BASIL variables)
  • Save and reuse SPARQL Results Sets files as input for parametric queries
  • Slice large CSV files with an iterator-like execution style (soon JSON and XML)
  • Supports an on-disk option (with Apache Jena TDB2)
  • Command line and server (via Jena Fuseki)
